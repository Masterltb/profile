**Giai đoạn 1: Nền tảng AWS**

- **[Tuần 1: Nền tảng AWS](1.1-week1/)**  
  Làm quen với AWS Console, CLI và các dịch vụ cơ bản (VPC, EC2).
- **[Tuần 2: Bảo mật & Chi phí](1.2-week2/)**  
  IAM Users/Roles, MFA, Cảnh báo chi phí (Billing Alerts).
- **[Tuần 3: IAM Nâng cao](1.3-week3/)**  
  Quyền tối thiểu (Least Privilege), EC2 Instance Profiles, Ngân sách.
- **[Tuần 4: Mạng (VPC)](1.4-week4/)**  
  Subnets, Route Tables, NAT Gateways, NACLs vs Security Groups.
- **[Tuần 5: Điện toán (EC2)](1.5-week5/)**  
  Khởi tạo instance, Custom AMIs, Session Manager, Lightsail.
- **[Tuần 6: Cơ sở dữ liệu (RDS)](1.6-week6/)**  
  Cài đặt RDS, Tính sẵn sàng cao (Multi-AZ), Read Replicas, Backup.
- **[Tuần 7: Tự động hóa](1.7-week7/)**  
  Viết script với AWS CLI v2, thiết lập môi trường Cloud9.
- **[Tuần 8: Kiểm tra](1.8-week8/)**  
  Ôn tập Well-Architected Framework và thi chứng chỉ **SAA-C03**.

**Giai đoạn 2: Triển khai Dự án "Hỗ trợ Cai thuốc lá"**

- **[Tuần 9: Khởi động Dự án](1.9-week9/)**  
  Khởi tạo **Program Service** (Java 25), Thiết kế DB, Logic Đánh giá.
- **[Tuần 10: Logic Cốt lõi](1.10-week10/)**  
  Triển khai Nhiệm vụ hàng ngày, Theo dõi Chuỗi (Streak), Xử lý sự kiện hút thuốc.
- **[Tuần 11: Tính năng Nâng cao](1.11-week11/)**  
  Logic Slip vs Relapse, Lập lịch Đánh giá tuần, Admin APIs.
- **[Tuần 12: Hoàn thiện](1.12-week12/)**  
  Kiểm thử Unit/Integration, Tối ưu hóa truy vấn, Tài liệu & Bàn giao.
