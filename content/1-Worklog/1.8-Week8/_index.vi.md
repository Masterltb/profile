### Kết quả đạt được

- Nắm vững 4 nhóm nội dung chính cần thiết cho kỳ kiểm tra.
- Kết nối các labs thực hành OJT (Tuần 2-7) với các mẫu kiến trúc và tình huống thi.
- Hiểu AWS Well-Architected Framework: Security, Reliability, Performance Excellence, Cost Optimization.
- Chuẩn bị ôn tập toàn diện cho ngày 31/10/2025.
- Sẵn sàng áp dụng kiến thức AWS để đưa ra quyết định kiến trúc thực tế.

**Kiến thức chính:**

- **Bảo mật:** IAM roles → least privilege | Encryption (KMS/TLS) → data protection | VPC security → network isolation | MFA → access control
- **Linh hoạt:** Multi-AZ → failover | Auto Scaling → elasticity | Load Balancers → traffic distribution | Route 53 → DNS failover
- **Hiệu năng:** Auto Scaling → handle spikes | Lambda → serverless compute | S3/EFS/EBS → storage optimization | Caching → reduced latency
- **Chi phí:** Reserved Instances → 30-70% savings | Budgets → spending control | Lifecycle policies → storage optimization | Right-sizing → efficient resources
